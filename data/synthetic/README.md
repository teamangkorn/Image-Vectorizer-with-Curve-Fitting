# Synthetic data

PGM (P5) test images generated by `build/gen_synthetic` and consumed
directly by `build/vectorizer`.  They are recreated on demand by
`experiments/benchmark.sh`, `tests/test_algorithms.sh`, and
`demo/cli_demo.sh`, so this directory is intentionally git-ignored
apart from this README.

Each image is a filled disk with a smaller concentric circular hole.
The Moore-Neighbour tracer therefore returns *two* closed contours
(outer + hole) with a combined raw vertex count of approximately
`8 r` (where `r` is the disk radius), giving us full control over the
input size used by the scaling experiment.
